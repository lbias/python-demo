Virtual Environment¶

Step1, Install python3¶

  • Method 1 (recommendation)

Go to Python Official Website<https://www.python.org/downloads/>, download and install a specific release.

  • Method 2

Use HomeBrew, $ brew install python3

Make sure Python3 has been installed successfully, $ python3 -V

Step2, Install package virtualenvwrapper¶

  1. $ pip3 install virtualenvwrapper, http://virtualenvwrapper.readthedocs.io/en/latest/
  1. check the package location, $ pip3 show -f virtualenvwrapper
  1. setup the path for shell zsh
  1. create a new environment, $ mkvirtualenv env1
  1. show the existing environment, $ workon, or $ lsvirtualenv
  1. use an existing environment, or switch between environments, $ workon env1
  1. You must use deactivate before removing the current environment, $ deactivate
  1. remove an environment, $ rmvirtualenv env1
